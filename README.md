# KotlinForStatement
for文

## 処理
1月から12月までを出力します。

## コード
```
fun main(args: Array<String>) {
    for (i in 1..12) println("${i}月")
}
```

## 出力結果  
```
1月
2月
3月
4月
5月
6月
7月
8月
9月
10月
11月
12月
```
